# 반응형 뷰포트

## 반응형 뷰포트 기점

| Extra Small | Small | Medium | Large | Extra Large |
| :---------: | :---: | :----: | :---: | :---------: |
|    349px    | 575px | 767px  | 991px |   1199px    |

<br />

### 사용 예시 (with Emotion)

```css
// Extra Large
@media (max-width: 1199px) {
}

// Large
@media (max-width: 991px) {
}

// Medium
@media (max-width: 767px) {
}

// Small
@media (max-width: 575px) {
}

// Extra Small
@media (max-width: 349px) {
}
```
