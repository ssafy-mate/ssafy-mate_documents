# 팀 관련

## 프로젝트 선택 시 (팀 생성 페이지)

| TYPE |         URL         | TOKEN |
| :--: | :-----------------: | :---: |
| GET  | /api/auth/user/team |   O   |

### Request

```json
{
  "selectedProject": "특화 프로젝트"
}
```

> params

### Response

- 팀이 없을 경우

```json
{
  "belongToTeam": true
}
```

- 팀이 이미 속해 있을 경우

```json
{
  "belongToTeam": false
}
```

- 서버 오류

```json
{
  "status": 500,
  "success": false
}
```

## CREATE | 팀 생성 요청

| TYPE |      URL       | TOKEN |
| :--: | :------------: | :---: |
| POST | /api/auth/team |   O   |

### Request

```json
{
  "teamImg": "팀 이미지.png(파일 형태)",
  "campus": "서울",
  "project": "특화 프로젝트",
  "projectTrack": "빅데이터",
  "teamName": "싸피메이트",
  "notice": "최우수상에 도전할 팀원을 모집합니다.",
  "introduction": "환상의 호흡으로 프로젝트 진행해 보고 싶습니다. 백엔드는 ... 프론트는 ... 이런 식으로 개발 진행하려고 합니다.",
  "techStacks": [1, 5, 8, 20, 42],
  "totalRecruitment": 6,
  "frontendRecruitment": 3,
  "backendRecruitment": 3
}
```

> 선택 항목 : introduction

> 최소 요건 : techStacks 2가지 이상

> formData 형식으로 전송

### Response

- 팀 생성 성공

```json
{
  "teamId": 1,
  "message": "팀을 성공적으로 생성하였습니다."
}
```

- 팀 생성 실패 (서버 문제)

```json
{
  "status": 500,
  "success": false,
  "message": "Internal Server, 팀 생성 실패"
}
```

## READ | 팀 리스트 요청

| TYPE |             URL             | TOKEN |
| :--: | :-------------------------: | :---: |
| get  | /api/auth/project/team-list |   O   |

### Request

```json
{
  "campus": "서울",
  "project": "특화 프로젝트",
  "project_track": "인공지능",
  "techstack_code": 1,
  "team_name": "싸피",
  "job1": "프론트엔드(Front-end)",
  "exclusion": false,
  "sort": "recent",
  "page": 1
}
```

> Query String Params 형식으로 요청

> exclusion : true 일 때 모집 마감 팀 공고 제외

> sort : 'recent' 일 때 등록 최신순 정렬, 'headcount' total 인원 적은 순 정렬

### Response

```json
{
  "teams": [
    {
      "teamId": 1,
      "teamImgUrl": "C:\\image\\team_image\\default_img.jpg",
      "campus": "서울",
      "project": "특화 프로젝트",
      "projectTrack": "인공지능",
      "teamName": "팀 싸피",
      "notice": "개발개발",
      "techStacks": [
        {
          "id": 1,
          "techStackName": "HTML5"
        },
        {
          "id": 2,
          "techStackName": "CSS3"
        }
      ],
      "totalRecruitment": 6,
      "frontendRecruitment": 3,
      "backendRecruitment": 3,
      "totalHeadcount": 1,
      "frontendHeadcount": 1,
      "backendHeadcount": 0,
      "createDateTime": null,
      "isRecruiting": true
    }
  ],
  "totalPage": 1,
  "nowPage": 1,
  "totalElement": 20
}
```

> isRecruiting : true 일 때 모집 중 false 일 때 모집 마감

- 서버 처리 실패

```json
{
  "status": 500,
  "success": false,
  "meesage": "Internal Server, 팀 리스트 조회 실패"
}
```
