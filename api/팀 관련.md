# 팀 관련

## 프로젝트 선택 시 (팀 생성 페이지)

| TYPE |         URL         | TOKEN |
| :--: | :-----------------: | :---: |
| GET  | /api/auth/user/team |   O   |

### Request

```json
{
  "selectedProject": "특화 프로젝트"
}
```

> params

### Response

- 팀이 없을 경우

```json
{
  "belongToTeam": true
}
```

- 팀이 이미 속해 있을 경우

```json
{
  "belongToTeam": false
}
```

- 서버 오류

```json
{
  "status": 500,
  "success": false
}
```

## CREATE | 팀 생성 요청

| TYPE |      URL       | TOKEN |
| :--: | :------------: | :---: |
| POST | /api/auth/team |   O   |

### Request

```json
{
  "teamImg": "팀 이미지.png(파일 형태)",
  "campus": "서울",
  "project": "특화 프로젝트",
  "projectTrack": "빅데이터",
  "teamName": "싸피메이트",
  "notice": "최우수상에 도전할 팀원을 모집합니다.",
  "introduction": "환상의 호흡으로 프로젝트 진행해 보고 싶습니다. 백엔드는 ... 프론트는 ... 이런 식으로 개발 진행하려고 합니다.",
  "techStacks": [
    {
      "techStacksName": "JavaScript"
    },
    {
      "techStacksName": "JPA"
    },
    {
      "techStacksName": "Spring Boot"
    }
  ],
  "totalRecruitment": 6,
  "frontendRecruitment": 3,
  "backendRecruitment": 3
}
```

> 선택 항목 : introduction

> 최소 요건 : techStacks 2가지 이상

> formData 형식으로 전송

### Response

- 팀 생성 성공

```json
{
  "message": "팀을 성공적으로 생성하였습니다."
}
```

- 팀 생성 실패 (서버 문제)

```json
{
  "status": 500,
  "success": false,
  "message": "Internal Server, 팀 생성 실패"
}
```

  "techStacks": [
    {
      "techStacksName": "JavaScript"
    },
    {
      "techStacksName": "JPA"

    {

> 선택 항목 : introduction

### Response

  "status": 400,

## DELETE | 팀 삭제

|  TYPE  |          URL           | TOKEN |
| :----: | :--------------------: | :---: |
| DELETE | /api/auth/team/:teamId |   O   |

### Request

```json
{}
```

> params

### Response

- 팀 삭제 성공

```json
{
  "meesage": "팀 삭제가 완료되었습니다."
}
```

- 권한 없을 때 (오너 정보 불일치)

```json
{
  "status": 400,
  "success": false,
  "meesage": "사용자는 팀 삭제 권한이 없습니다."
}
```

- 서버 처리 실패

```json
{
  "status": 500,
  "success": false,
  "meesage": "Internal Server, 팀 삭제 실패"
}
```
