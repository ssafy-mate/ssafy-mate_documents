# 로그인 페이지 API

## 로그인

| TYPE |       URL        |
| :--: | :--------------: |
| POST | /api/user/signin |

- Request

```json
{
  "email": "ssafy@test.com",
  "password": "a12345"
}
```

- Response

```json
{
  "status": 200,
  "success": true,
  "message": "",
  "token": "ad123sdafgfa0asdfas12390",
}

{
  "status": 401,
  "success": false,
  "message": "아이디 또는 비밀번호를 확인하세요.",
}
```
