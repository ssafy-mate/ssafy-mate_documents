# 로그인 페이지 API

## 로그인

| TYPE |        URL        | TOKEN |
| :--: | :---------------: | :---: |
| POST | /api/user/sign-in |   X   |

### Request

```json
{
  "email": "ssafy@test.com",
  "password": "a12345"
}
```

### Response

- 로그인 성공

```json
{
  "status": 200,
  "success": true,
  "message": "",
  "token": "ad123sdafgfa0asdfas12390"
}
```

- 로그인 실패 (아이디 또는 비밀번호 불일치)

```json
{
  "status": 401,
  "success": false,
  "message": "아이디 또는 비밀번호가 잘못 입력 되었습니다."
}
```
