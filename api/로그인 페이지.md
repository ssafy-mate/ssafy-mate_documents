# 로그인 페이지 API

## 로그인

| TYPE |        URL        | TOKEN |
| :--: | :---------------: | :---: |
| POST | /api/user/sign-in |   X   |

### Request

```json
{
  "userEmail": "ssafy@test.com",
  "password": "a12345"
}
```
