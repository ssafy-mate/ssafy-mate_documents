# 홈 페이지

## 프로젝트 트랙 선택

| TYPE |           URL           | TOKEN |
| :--: | :---------------------: | :---: |
| POST | /api/user/project/track |   O   |

### Request

```json
{
  "project": "특화 프로젝트",
  "projectTrack": "빅데이터"
}
```

```json
{
  "project": "공통 프로젝트",
  "projectTrack": "웹 기술"
}
```

### Response

- 트랙 선택 성공

```json
{
  "success": true,
  "message": "특화 프로젝트 트랙 선택이 완료되었습니다."
}
```

```json
{
  "success": true,
  "message": "공통 프로젝트 트랙 선택이 완료되었습니다."
}
```

- 유효하지 않는 토큰

```json
{
  "status": 401,
  "success": false,
  "message": "토큰이 유효하지 않습니다."
}
```

- 이미 트랙을 선택한 사용자일 때

```json
{
  "status": 409,
  "success": false,
  "message": "이미 트랙 선택을 완료하였습니다."
}
```

- 인증 실패 (서버 오류)

```json
{
  "status": 500,
  "success": false,
  "message": "Internal Server Error, 교육생 인증 실패"
}
```
