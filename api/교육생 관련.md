## UPDATE | 교육생 상세 정보 수정 요청

| TYPE |                 URL                  | TOKEN |
| :--: | :----------------------------------: | :---: |
| PUT  | /api/auth/users/:userId/:profileInfo |   O   |

**profileInfo** : ssafy-track, profileImg, self-introduction, jobs, tech-stacks, urls

### Request

```json
{
  "ssafyTrack": "Java Track"
}
```

```json
{
  "profileImg": "프로필 사진"
}
```

```json
{
  "selfIntroduction": "수정했어요..."
}
```

```json
{
  "job1": "프론트엔드 (Front-end)",
  "job2": "백엔드 (Back-end)"
}
```

```json
{
  "techStacks": [
    {
      "techStackCode": 1,
      "techStackLevel": "상"
    },
    {
      "techStackCode": 2,
      "techStackLevel": "중"
    },
    {
      "techStackCode": 3,
      "techStackLevel": "중"
    },
    {
      "techStackCode": 4,
      "techStackLevel": "중"
    },
    {
      "techStackCode": 5,
      "techStackLevel": "중"
    }
  ]
}
```

```json
{
  "githubUrl": "https://github.com/ssafy-mate",
  "etcUrl": "https://velog.io/@ssafy-mate"
}
```

> formData 형식으로 전송
>
> 최소 요건 : techStacks 2가지 이상

### Response

- 교육생 상세 정보 수정 성공 시

```
{
  "message": "교육 트랙/프로필 이미지/자기 소개/희망 직무/기술 스택/SNS 링크 수정이 완료되었습니다."
}
```

- 교육생 상세 정보 수정 권한 없음

```
{
  "status": 403,
  "success": false,
  "message": "사용자는 정보를 수정할 수 있는 권한이 없습니다."
}
```

- 교육생 상세 정보 수정 실패 시

```
{
  "status": 500,
  "success": false,
  "message": "Internal Server, 교육생 상세 정보 수정 실패"
}
```

## READ | 교육생 프로젝트 정보 받기

| TYPE |               URL                | TOKEN |
| :--: | :------------------------------: | :---: |
| GET  | /api/auth/users/:userId/projects |   O   |

### Request

```json
{}
```

### Response

- 프로젝트 정보 받기 성공

```json
"projects": [
  {
    "projectId": 1,
    "projectName": "공통 프로젝트",
    "projectTrack": "웹 기술",
    "projectTeamId": 10
  },
  {
    "projectId": 2,
    "projectName": "특화 프로젝트",
    "projectTrack": null,
    "projectTeamId": null,
  },
  {
    "projectId": 3,
    "projectName": "자율 프로젝트",
    "projectTrack": null,
    "projectTeamId": null,
  },
```

- 토큰 유효하지 않을 시

```json
{
  "status": 403,
  "success": false,
  "message": "내 프로젝트 정보 조회 권한이 없습니다."
}
```

- 서버 오류

```json
{
  "status": 500,
  "success": false,
  "message": "Internal Server Error, 내 프로젝트 정보 갱신 실패"
}
```

## READ | 교육생 팀 아이디 조회

| TYPE |               URL               | TOKEN |
| :--: | :-----------------------------: | :---: |
| GET  | /api/auth/users/:userId/belong-to-team |   O   |

```json
{
  "project": "특화 프로젝트"
}
```

### Response

```json
{
  "status": 500,
  "success": false,
  "meesage": "Internal Server, 팀 합류 여부 조회 실패"

```
